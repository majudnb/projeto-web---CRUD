Tema do projeto: Sistema de Cadastro de Clientes e Livros

Descrição do projeto:
Este projeto foi desenvolvido em PHP com MySQLi e tem como objetivo cadastrar, listar, alterar e excluir registros de clientes e livros.
Ele é uma continuação do CRUD de Clientes feito em sala, adicionando agora o módulo de livros, que combina com o tema original.

O que é CRUD?
CRUD são quatro operações básicas realizadas em um banco de dados:

C – Create (Criar)

R – Read (Ler)

U – Update (Atualizar)

D – Delete (Excluir)



Desafios implementados:
- Criação de um segundo CRUD completo (Livros).

- Organização dos arquivos em pastas (raiz e admin).

- Páginas institucionais: Home, Quem Somos e Fale Conosco.

- Botão para voltar ao menu após o cadastro.

- Adição de CSS para deixar o sistema mais bonito.



Como rodar o projeto:

1.Abra o XAMPP e inicie o Apache e o MySQL.

2.No phpMyAdmin, crie um banco de dados chamado crud_livros.

3.Importe o arquivo crud_livros.sql que está junto com o projeto.

4.Copie a pasta crud_livros para dentro de:
C:\xampp\htdocs\

5.No navegador, acesse:
http://localhost/crud_livros/



Resumo da entrega:

- Projeto completo em ZIP

- Arquivo SQL do banco de dados

- Arquivo README.txt
